福昕PDF阅读器 V4去广告方法

Posted on 2011-03-07 | In 默认分类

清理历史痕迹的批处理

将以下批处理内容（蓝色字体），复制到记事本，另存为，“清理Foxit PDF Reader历史痕迹.bat”即可。

（名字任意，不带双引号）

———————————-华丽的分割线———————————————

@echo off

title清理Foxit PDF Reader历史痕迹

echo.

echo清理Foxit PDF Reader历史痕迹

pause

reg delete “HKCU\Software\Foxit Software\福昕阅读器\Recent File List” /f >nul 2>nul

echo.

echo.

echo.

echo清理完成

echo.

echo按任意键关闭

pause>nul

exit

——————————–华丽的分割线———————————————–

设置为不显示广告的批处理

将以下批处理内容（绿色字体），复制到记事本，另存为，“不显示软件界面右上角的广告条.bat”即可。

（名字任意，不带双引号）

——————————–华丽的分割线———————————————–

@echo off

title不显示软件界面右上角的广告条

echo.

echo不显示软件界面右上角的广告条

pause

reg add “HKCU\Software\Foxit Software\福昕阅读器\MainFrame” /f /v ShowAd /t REG_SZ /d 0 >nul 2>nul

echo.

echo.

echo.

echo设置完成

echo.

echo按任意键关闭

pause>nul

exit

——————————–华丽的分割线———————————————–
